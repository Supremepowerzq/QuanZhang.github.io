具身智能（Embodied AI）医疗介入手术机器人系统

2024.11 — 至今

第一作者发表 Cell 子刊 Device（IF: 8.0）论文 1 篇；共同作者发表 iScience（IF: 4.1）论文 1 篇；申请国家发明专利 2 项、软件著作权 3 项

论文 / 代码

论文题目：Addressing the "world map problem" for submillimeter medical robots requires advancing local sensing beyond external imaging dependence；DOI： 10.1016/j.device.2025.100839

面向微创介入临床需求，研发手术机器人端到端具身智能控制系统，完成仿真训练到真机部署的全栈算法开发。基于 PyTorch + Gym + Stable-Baselines3 + MuJoCo 搭建开发框架，实现视觉/力觉/位姿多模态数据融合，构建感知—决策—控制全闭环链路。

在算法层面，搭建模仿学习预训练 + PPO 强化学习微调两阶段训练流水线；通过域随机化与残差网络实现 Sim-to-Real 高鲁棒迁移；集成 Depth-Anything-V2 深度估计与 U-Net 目标检测，支持术中实时视觉感知；基于 TensorRT 完成模型量化部署，实现临床毫秒级推理响应。

Research

Embodied-AI Medical Interventional Surgical Robotic System

2024.11 – Present

One first-author paper in Device (Cell Press, IF: 8.0); co-author in iScience (IF: 4.1); 2 invention patents; 3 software copyrights

Paper / Code

Paper title: Addressing the "world map problem" for submillimeter medical robots requires advancing local sensing beyond external imaging dependence; DOI: 10.1016/j.device.2025.100839

For minimally-invasive interventional clinical needs, we develop an end-to-end embodied intelligence control system for surgical robots, completing full-stack algorithm development from simulation training to real-machine deployment, achieving precise autonomous soft robotic manipulation in complex luminal environments. Built on PyTorch + Gym + Stable-Baselines3 + MuJoCo, integrating vision/force/pose multimodal data fusion.

We establish a two-stage training pipeline (imitation learning pretraining + PPO fine-tuning); achieve robust Sim-to-Real transfer via domain randomization and residual dynamics modeling; integrate Depth-Anything-V2 and U-Net for real-time visual perception; deploy quantized models via TensorRT for clinical millisecond-level inference.

Autonomous Target-Tracking Interventional Surgical Robot

2023.8 – Present

IROS 2025 paper; 2 invention patents; 2 software copyrights; Grand Prize, China–Germany Robotics Competition

Paper / Code

Paper title: Continuous renal calculi tracking for autonomous robotic ureteroscopic lithotripsy; DOI: 10.1109/IROS60139.2025.11247411

Addressing challenges of occlusion, motion blur, and tissue deformation during ureteroscopic lithotripsy, we design spatiotemporal perception and bias-suppression strategies achieving ≈0.3 mm localization error. We enhance U-Net for clinical image segmentation, integrate deep networks with DRL controllers for robust manipulation, and build high-fidelity simulation for policy training.
